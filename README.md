# 文豪.skill

> *作家を、1つのプロンプトへと蒸留します。*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Fork of](https://img.shields.io/badge/fork%20of-nuwa--skill-lightgrey)](https://github.com/alchaincyf/nuwa-skill)

<br>

**太宰治、三島由紀夫、カフカ、村上春樹——あるいは、あなたの好きな個人ブロガーでも。**

作品集を渡せば、文豪.skill は 14 層 × 5 カテゴリの文体 DNA を抽出し、
1 つの skill へと蒸留します。skill をロードすれば、AI は**その作家になります**。
模倣ではなく、憑依。

[使用例](#使用例) · [インストール](#インストール) · [動作原理](#動作原理)

## 使用例

### 書く — 作家の筆で文章を生成する

```
ユーザー ❯ 太宰治.skill をロードして、
           「昨日会社を休みました。申し訳ありません」を書き直して

太宰治   ❯ 昨日、出社できませんでした。恥ずかしい話で御座います。
           申し訳御座いません、など口にするのも、今更な気が致します。
           そもそも、私という人間が会社に行くという行為そのものが、
           何やら滑稽な茶番のようにさえ、思われてならぬのです。
```

### 添削 — 作家があなたの原稿を手直しする

```
ユーザー ❯ 三島由紀夫.skill をロードして、この冒頭を添削して
           「月が綺麗だった。彼女は泣いていた。」

三島   ❯  [赤入れ]
           月は、鋼の切先を思わせる白光を放つて中天に懸かつてゐた。
           彼女の頬に、その冷光が一筋の流れを描いた。

           [指摘]
           君は情緒を急ぎ過ぎる。叙景とは、まず物体の量感・温度・光量を
           読者に手渡し、感情はその後にしか来ぬものだ。
```

### 対話 — 作家の視点で相談・批評する

```
ユーザー ❯ カフカ.skill をロードして、キャリアの相談をしたい

カフカ   ❯ あなたの勤める会社のことを、まずお聞きしたい。
           その会社の建物は、内側からしか入れぬ構造をしていませんか。
           外に出たと思った瞬間、入口に立っている、というような。
           そうだとしたら、問題は会社ではなく、扉の定義のほうです。
```

## 動作原理

skill をロードすると AI は「作家について語る人」ではなく「**その作家**」になる。これを可能にするのは、印象批評ではなく構造記述——「太宰治は叙情的」ではなく「主語省略率 72%、1 文平均 28 字、自己否定→自己陶酔の対句を 20 文に 1 回」。

蒸留は**5 カテゴリ・14 層**で作家の言語的生理を捕捉する：

| カテゴリ | 層 |
|---|---|
| **聲** Voice | L1 語彙 DNA · L2 構文 · L3 音韻・表記 |
| **眼** Eye | L4 視点・時制 · L5 五感 · L6 読者距離 |
| **骨** Bones | L7 段落・呼吸 · L8 対話 · L9 物語構造 |
| **魂** Soul | L10 主題・世界観 · L11 レトリック · L12 人格・心理 |
| **界** Boundary | L13 反パターン · L14 限界 |

入力は**作品集（最優先）→ Wikipedia（必須補助）→ Google（任意）**の順。交付前に独立した敵対 agent が 100 点満点で採点し、**90 点以上**でなければ配布しない。

## インストール

```bash
npx skills add illusions-lab/bungo-skill
```

Claude Code で：

```
> 太宰治を蒸留して
> 村上春樹の skill を作って
> カフカを蒸留してほしい、作品集は sources/ に置いた
```

### 蒸留済みの作家（独立リポジトリ）

| 作家 | リポジトリ | 状態 |
|------|--------|----|
| 太宰治 | [illusions-lab/dazai-skill](https://github.com/illusions-lab/dazai-skill) | 🎉 完了（作品集 278 篇、約 293 万字） |

各 skill は独立リポジトリとして発行され、`npx skills add` で単独インストールできる。
