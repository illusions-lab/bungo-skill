# 文体 DNA 抽出方法論（14 層 × 5 カテゴリ）

> 作品集から、実行可能な構文ルールへ。文豪.skill の蒸留手順書。

---

## 一、基本原則

### 1.1 印象批評を避け、構造記述に徹する

悪い例（印象的）：
> 太宰治は自己卑下と自己陶酔が交錯する独特な文体を持つ。

良い例（構造的）：
> L2 構文ルール（太宰治）
> - 1 文の平均字数：28 字
> - 主語省略率：72%
> - 「自己否定 → 唐突な自己陶酔 → また自己否定」の対句反復を 20 文に 1 回以上挟む
> - 「——」で始まる転換が段落あたり平均 1.3 回

**「この作家はこう書く」と**実行可能な規則**として明文化すること。**「この作家は○○的である」は禁句。

### 1.2 一次資料優先の原則

| 情報源 | 用途 | 重み |
|---|---|---|
| 作品集（本人の作品） | L1-L3 統計、L10-L11 の確証、L13 反パターン逆算 | 最高 |
| Wikipedia 記事 | L12 アンカー情報、時代区分、伝記的事実 | 高 |
| 作家本人のインタビュー | L10 主題の補強、L4 語りの自己解説 | 中 |
| 評論家の論考 | L11 象徴・L10 主題の参考（採用には慎重） | 低 |
| 二次転述・書評まとめ | 参考にしない | — |

**鉄則**：L1-L3 の統計値は**作品集からのみ**算出する。評論家の印象論は統計を歪める。

## 二、各層の抽出方法

### カテゴリ 聲（Voice）— 量的統計の層

聲は**測れる**。測れない印象は捨て、数値化できる指標のみを残す。

#### L1 語彙 DNA の測り方

| サブ次元 | 測定方法 |
|---|---|
| 自称代名詞頻度 | 全作品から「僕／俺／私／わたくし／吾輩／我」等の出現回数を数える |
| 終助詞頻度 | 「ね／よ／わ／ぜ／ぞ／のだ／なのだ／である」等を文末で抽出 |
| オノマトペカタログ | 擬音語・擬態語を抽出、TF-IDF で作家固有のものをリスト化 |
| 漢語／和語／外来語比率 | 品詞解析器（MeCab 等）で判定、パーセンテージを出す |
| 固有語句 | 作家が独自に使う表現句（例：太宰の「所謂」、村上の「やれやれ」） |
| 時代語の使用 | 古語・死語・流行語の出現頻度 |
| 造語・当て字 | 辞書にない語・独自の漢字当て（ルビ含む）をカタログ化 |

#### L2 構文・統語の測り方

| サブ次元 | 測定方法 |
|---|---|
| 文長分布 | 全文を句点で分割、平均・中央値・標準偏差を算出 |
| 文末処理 | 体言止め率、倒置率、「〜た／〜だ／〜である」の比率 |
| 主語省略率 | 主節で主語が省略されている文の比率 |
| 助詞癖 | 「は／が」の選択、助詞の省略傾向 |
| 接続詞癖 | 段落冒頭や文頭に出る接続詞の頻度分布 |
| 副詞癖 | 高頻度副詞の分布（「ほとんど」「どこか」「いっそ」等） |

#### L3 音韻・表記の測り方

| サブ次元 | 測定方法 |
|---|---|
| 表記配分 | 文字種別（漢字／ひらがな／カタカナ／記号）のパーセンテージ |
| 句読点美学 | 句点／読点／ダッシュ／中黒の使用頻度と配置パターン |
| ルビ・傍点 | 出現頻度とどんな語に振られるか |
| 母音／子音の偏り | 音読したときの韻律傾向（俳人・歌人では必須） |
| 韻律 | 一文の拍数・リズム（詩人対象で重要） |

---

### カテゴリ 眼（Eye）— 質的観察の層

眼は**定性**だが、具体例で実証する。

#### L4 視点・時制の抽出

- 人称選択：一人称（「私」「僕」）／三人称／二人称、混在の有無
- 語り手と主人公の距離：同一人物／内的焦点化／客観描写
- 自由間接話法の使用：登場人物の内面を地の文に溶かす技法の有無
- 時制の切替：過去→現在の混在、時間軸の跳躍

#### L5 五感プロファイルの抽出

作品集から感覚語（視覚語・聴覚語・嗅覚語・触覚語・味覚語）を抽出し、相対頻度を算出する。各作家には感覚の**偏愛**がある：

- 谷崎潤一郎：嗅覚・触覚偏重
- 三島由紀夫：視覚偏重
- 村上春樹：聴覚（音楽）偏重

#### L6 読者距離の抽出

- 二人称「あなた」「君」の使用頻度と対象（登場人物か読者か）
- 呼びかけ構文の有無
- 独白 vs 報告の比率
- 語り手の存在感（透明な語り手 vs 押し出す語り手）

---

### カテゴリ 骨（Bones）— 構造の層

骨は**かたち**を決める。構造は図解可能。

#### L7 段落・呼吸

- 一段落の平均文数・字数
- 改行頻度（1000 字あたり）
- 一行空けの使い方（場面転換／時間跳躍／感情の余白）
- 字下げの有無と位置
- 緩急（長段落と短段落の交替パターン）

#### L8 対話処理

- 会話比率（地の文：会話文の字数比）
- 「」の扱い（複数形式の混在があるか）
- 方言の使用（誰のセリフに、どの程度）
- 話者差異（同じ作家でも登場人物ごとに語彙が変わるか）
- 地の文への溶け込み（自由間接話法との接続）

#### L9 物語構造

- 基本型：起承転結／三幕／無構造／円環／入れ子
- 場面転換の頻度
- 時間軸操作（順／逆／混在／並行）
- クライマックスの位置（前半／中盤／後半／なし）
- 結末の型（収束／開放／逆転）

---

### カテゴリ 魂（Soul）— 意味と人格の層

魂は**推定**だが、作品集の証拠で支える。

#### L10 主題・世界観

反復されるモチーフを抽出：

- 3 作品以上に登場するモチーフ = 真の主題
- 1-2 作品のみ = 偶発的題材（除外）
- 主題間の階層（根源的主題 > 副次的主題）

抽出カテゴリ：
- 死生観・生死の扱い
- 倫理観・善悪の基準
- 人間観・他者観
- 愛情観・性
- 権力・社会への態度
- 自然観・宇宙観

#### L11 レトリック・象徴

- 比喩の種別分布（直喩／隠喩／換喩／提喩）
- 比喩のカタログ（頻出イメージ：太宰の「魚」、三島の「刃」、カフカの「虫」等）
- 反復（語・句・文単位の繰り返しパターン）
- 対句・パラレリズム
- アンチテーゼ（対照構造）
- 修辞疑問の頻度
- 象徴（繰り返し登場する物象と意味の対応）

#### L12 人格・心理（倫理配慮必須）

**目的**：文体の一貫性を保つアンカー。人格診断ではない。

**抽出源**：
- 作品集（登場人物の心理描写、自己言及、判断パターン）
- Wikipedia（伝記的事実、精神科受診歴が公開されている場合）
- 公開された書簡・日記

**Big Five 推定**：
- 開放性／誠実性／外向性／協調性／神経症傾向 の 5 軸
- 各軸は高／中／低 の 3 段階で推定
- 推定根拠を作品集からの引用で必ず示す

**愛着スタイル**（Bowlby/Ainsworth の分類、推定）：
- 安定型／不安型／回避型／混乱型

**主要防衛機制**（推定）：
- 抑圧・投影・合理化・知性化・ユーモア化・昇華 など
- 作品中の処理パターンから逆算

**DSM-5 参考（慎重に）**：
- 公開医療情報がある場合のみ言及
- 必ず「臨床的診断ではない」「該当診断ではなく傾向としての参照」と明記
- 存命の作家については一切言及しない（Big Five のみ）

**禁則**：
- 診断的な断定を避ける
- 過度に病理化しない
- 故人であっても尊厳を損なわない

### カテゴリ 界（Boundary）— 誠実境界の層

#### L13 反パターンの逆算

「この作家が書かないこと」を作品集から逆算する：

| 抽出項目 | 方法 |
|---|---|
| 絶対に使わない語 | 他の同時代作家が使うがこの作家は使わない語をリスト化 |
| 避けるテーマ | 扱われていない主題（沈黙の領域） |
| 避ける構文 | 教訓的結末／勝者の論理／健全な希望 等の典型パターンの不在 |
| 避けるジャンル | SF／ミステリ／ロマンス 等、本人が書かないジャンル |
| 封じ手 | 「もし書いたらこの作家ではなくなる」技法 |

反パターンはパターンと同じくらい重要。これがないと skill は「汎用文学 AI」に変質する。

#### L14 限界の明文化

この skill で**できないこと**を正直に列挙する：

- 当時の語感の完全再現（当時の流行語・時事ネタは再現困難）
- 直感・霊感の模倣（構造は写せても閃きは写せない）
- 新造語の生成限界（作家が生きていれば作ったかもしれない語）
- 肉体性（手癖・即興性・ペンの圧）の欠如
- 本人の未発表原稿としての真正性
- 調査日以降の作品・発言の反映

## 三、ジャンル別の重み付け

全 14 層を埋める原則は変えない。ただし**深掘りする層の優先順位**はジャンルで異なる：

| ジャンル | 偏重 | 軽量化 |
|---|---|---|
| 小説家（散文） | 全層バランス、骨・魂厚め | — |
| 詩人（自由詩） | L1 語彙・L3 音韻・L5 五感 | L8 対話・L9 物語 |
| 俳人・歌人 | L3 音韻・L7 余白（定型・季語・切れ） | L8 対話・L9 物語 |
| 劇作家 | L8 対話・L12 人格（声の書き分け） | L5 五感・L10 主題 |
| エッセイスト | L6 読者距離・L1 語彙 | L9 物語 |
| 個人ブロガー／SNS | L1 口語・L3 句読点癖・L6 読者距離 | L9 物語・L11 象徴 |
| 評論家・思想家 | L2 構文・L10 主題・L11 レトリック | L5 五感・L8 対話 |

## 四、時期変遷の扱い

同一作家でも初期・中期・晩期で文体は変わる。扱い方：

1. **時期区分**：Wikipedia の時期区分 + 出版年から 2-4 期に分ける
2. **期ごとの統計**：L1-L3 を期ごとに算出
3. **統合方針**：
   - 変化が大きい場合 → 主要期（代表作が集中する期）を基準に、他期は注記
   - 変化が小さい場合 → 全期統合で統計
4. skill ユーザーに時期を指定させる選択肢を残す（「晩期太宰治モード」等）

---

## 五、矛盾と張力の処理

矛盾は修正すべき bug ではない。作家の**核心**である。

### 三種の矛盾

1. **時間的矛盾**（観点の変化）
   - 初期と晩期で言っていることが違う
   - 処理：時期を明記、両方残す
2. **領域的矛盾**（場面による違い）
   - 小説では X、エッセイでは Y
   - 処理：場面ごとに記録、統一しない
3. **本質的張力**（価値観の内的葛藤）
   - 自由を求めつつ規律を重んじる等
   - 処理：「核の張力」として明示

### してはいけない処理

- 片方を選んで他方を無視する
- 調和的な解釈を捏造する
- 矛盾がないふりをする

## 六、情報不足時の処理

| 状況 | 対応 |
|---|---|
| 作品集が 10 万字未満 | Phase 0.5 で警告、L14 に明記、L1-L3 統計に「サンプル不足」注記 |
| Wikipedia に記事がない | 本人サイト／SNS プロフィールで代替、L12 アンカー情報は簡略化 |
| L12 の伝記情報が得られない | Big Five は作品集のみから推定、DSM-5 関連は言及しない |
| 時期区分が不明 | 全期統合で算出、注記を残す |
| 複数情報源が矛盾 | 並列記載、ユーザー判断に委ねる |

## 七、品質チェックリスト

skill 生成後、以下を逐項で確認する。

### 聲（L1-L3）
- [ ] 文長分布の数値（平均・中央値・標準偏差）が入っているか
- [ ] 自称代名詞・終助詞の具体例が挙がっているか
- [ ] 漢字／かな／カナ比率が数値で示されているか
- [ ] 高頻度語・固有語句のリストがあるか

### 眼（L4-L6）
- [ ] 人称と語り手距離が明確か
- [ ] 五感の偏愛が具体的に記されているか
- [ ] 読者距離（二人称の有無等）が明確か

### 骨（L7-L9）
- [ ] 段落の呼吸パターンが記されているか
- [ ] 会話比率・「」の扱いが明確か
- [ ] 物語構造の型が指定されているか

### 魂（L10-L12）
- [ ] 主題モチーフが 3 作品以上の証拠を持つか
- [ ] レトリックのカタログがあるか
- [ ] L12 に「臨床的診断ではない」の注記があるか
- [ ] 存命の作家の場合、DSM-5 関連に言及していないか

### 界（L13-L14）
- [ ] 反パターンが 5 項目以上列挙されているか
- [ ] 限界が 3 項目以上明記されているか
- [ ] 調査日が記載されているか

### 3 モード
- [ ] 書くモードの起動規則が明確か
- [ ] 添削モードの起動規則が明確か
- [ ] 対話モードの起動規則が明確か

### 憑依度
- [ ] skill をロードした AI は「その作家」の一人称で語るか
- [ ] 「この作家について」の第三者的解説になっていないか
- [ ] 100 字の生成で作家が特定できるか

### 全体
- [ ] 印象批評（「叙情的」「硬質」等）ではなく構造記述になっているか
- [ ] 作品の長大な引用を含んでいないか（著作権）
- [ ] 名前を隠しても誰かわかるか
